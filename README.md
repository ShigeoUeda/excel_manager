# excel_manager
Excelファイルのセルからデータ取得・格納するなどのライブラリ

```bash
# GitHubからリポジトリをクローン
git clone https://github.com/ShigeoUeda/excel_manager.git
cd excel_manager

# Pythonの仮想環境を作成・有効化
python -m venv venv
source venv/bin/activate 
# ライブラリのインストール
pip install -r requirements.txt

# サンプルの実行
python excel_manager.py -f sample.xlsx
```
